# HR-Performance-Analytics-Dashboard
An interactive performance dashboard designed to analyze HR metrics and visualize employee data. This project demonstrates data cleaning, pivot table analysis, and insight generation in Excel to drive informed HR decision-making.
The goal was to shift from simple data tracking to identifying trends that support better decision-making within the Human Resources department.
Key Objectives
• Data Cleaning: Standardized and organized raw employee records to ensure accuracy for reporting.
• Performance Tracking: Created pivot tables to identify key performance indicators (KPIs) such as department productivity and employee distribution.
• Visual Storytelling: Built a dynamic dashboard that allows stakeholders to quickly assess organizational performance at a glance.
Tools Used
• Microsoft Excel: Utilized for data cleaning, complex Pivot Table construction, and interactive dashboard visualization. The file also includes VBA macros to automate reporting workflows.
Key Insights
1. Overview: The current organization-wide attrition rate is 16.08%.
2. Departmental Performance: High attrition in specific departments, such as Research & Development or Sales, highlights a critical area for management focus.
3. Salary & Retention: A notable concentration of turnover appears in lower salary brackets, indicating that compensation structure may directly impact retention.
4. Operational Impacts: Business travel demands show a correlation with turnover patterns, suggesting that travel-heavy roles may lead to higher employee burnout.
